# 🧪 Studio Lab

A full-stack creative lab platform.

Built with **Next.js (App Router)**, **Tailwind CSS**, **NextAuth.js**, and tested with **Jest**.

## 🔧 Getting Started

```bash
git clone https://github.com/yourusername/studio-lab
cd studio-lab
pnpm install
pnpm dev
```
